# Create Province

To create the data in the specified collection with the specified parameters.
When a JSON object is saved, the object properties are automatically added to the collection you specified.

**Method :** <Badge>POST</Badge>

**URL :** <Badge variant="dark">/api/province</Badge>

**Body :** <Badge variant="purple">JSON</Badge>

```json
{
	"name": "New Province",
	"image_url": "https://images.unsplash.com/photo-1682687220063-4742bd7fd538?auto=format&fit=crop&q=60&w=500",
	"island_id": 1
}
```

<Param name='name' type='string' required={true}>
    Province name.
</Param>
<Param name="image_url" type="string" required={true}>
    Province image URL
</Param>
<Param name="island_id" type="number">
    Province island ID
</Param>

**Response :** <BadgeOutline variant="success">200 - OK</BadgeOutline>

```json
{
  "message": "Success add province"
}
```

**Response :** <BadgeOutline variant="info">422 - Unprocessable Content</BadgeOutline>

```json
{
    "error": [
        {
            "code": "too_small",
            "minimum": 1,
            "type": "string",
            "inclusive": true,
            "exact": false,
            "message": "Name is required",
            "path": [
                "name"
            ]
        },
        {
            "code": "too_small",
            "minimum": 1,
            "type": "string",
            "inclusive": true,
            "exact": false,
            "message": "Image URL is required",
            "path": [
                "image_url"
            ]
        }
    ]
}
```

**Response :** <BadgeOutline variant="danger">500 - Server Error</BadgeOutline>

```json
Internal Server Error
```
